# FL_Theme_Park_Weather
Analyzed historical Florida weather and tourism data using SQL to uncover patterns and insights. The analysis considers weekends and holidays to better understand how these factors influence visitor attendance at theme parks and other attractions.

#Project Overview

This project analyzes the relationship between Florida weather patterns and tourism activity across major theme parks and destinations. Using historical weather data and tourism statistics, the model identifies trends, correlations, and peak visitor periods, providing actionable insights for forecasting and operational planning.

#Key Features
	•	Integrated historical weather data (temperature, precipitation, humidity) with tourism and theme park attendance datasets.
	•	Developed SQL queries to extract insights on peak visitation days, seasonal trends, and weekend vs. weekday patterns.
	•	Modeled adjusted revenue and visitor counts accounting for holidays and weather impacts.
	•	Visualized findings using Python, creating charts to highlight trends and anomalies.
	•	Identified actionable insights for optimizing staffing, promotions, and visitor experience.

#Skills & Tools
	•	SQL for data extraction, aggregation, and conditional analysis.
	•	Python for data cleaning, visualization, and trend analysis.
	•	Data interpretation to understand the influence of environmental factors on tourism.
	•	Portfolio-ready presentation of findings using charts and tables for clarity.
